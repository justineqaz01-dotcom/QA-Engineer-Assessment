QA Engineer Technical Assessment

This assessment demonstrates manual testing, automation testing, and API testing capabilities.

-Web Application Testing
The SauceDemo website was tested for the following core flow:
Login → Add Product to Cart → Checkout → Complete Purchase.

-Manual test cases were created and executed to verify functionality

-Automation
Playwright was used to automate the main purchase flow of the application.

-API Testing
API endpoints from JSONPlaceholder were tested using Postman. Requests were validated using status codes and response structures.

- API Testing Documentation
